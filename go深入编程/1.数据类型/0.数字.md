1. 在go 语言中数字类型分为整形和浮点型

2. 整形 分为无符号和有符号

+ 有符号，表示数字有正负之分

    + int: 根据操作系统来区分，如果操作系统是32 位的，对应就是int32 类型，如果是64 位，就是int64 类型

    + int8:

    + int16

    + int32

    + int64

+ 无符号

    + uint8

    + uint16

    + uint32

    + uint64

    + uint: 根据操作系统来区分，如果操作系统是32 位的，对应就是int32 类型，如果是64 位，就是int64 类型

3. 浮点数： 在go 语言中浮点数只有float32 和 float64 ，默认是float64
