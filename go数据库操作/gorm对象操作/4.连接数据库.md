1. gorm 连接数据库需要相应的数据库驱动，以mysql 为例

        安装gorm 和 MySQL 驱动
        // 安装gorm
        go get -u gorm.io/gorm
        // 安装mysql 驱动
        go get -u gorm.io/driver/mysql
