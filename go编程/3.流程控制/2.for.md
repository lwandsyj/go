1. 在go 语言中循环只有for ，但是for 有很多变体

+ for 无限循环

        for {
            没有查询条件，会一值循环

        }

+ for 模拟while

        for left<right{
            // 当左边一直小于右边时
        }

+ for 正常循环

        for i:=0;i<len(arr);i++{

        }

        for i:=len(arr)-1;i>=0;i--{

        }
