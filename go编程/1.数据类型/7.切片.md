1. 切片和数组差不多，可以把切片看成动态数组，切片的长度是可以增加的

2. 切片是引用类型，底层使用的是数组

3. 切片声明定义和数组差不多

        var arr []int // [] 这里面没有长度

4. 返回切片的长度

        len(arr)

5. 给切片添加元素，使用append 在切片后面添加元素

        arr = append(arr,2)

6. go 没有中没有提供切片的删除操作，和其他添加元素的操作，只能通过在切片

        a:=arr[0:3] // 对原切片的引用

        a=append(a,3)

        a=append(a,arr[3:]...)

        a// 为添加后的切片
